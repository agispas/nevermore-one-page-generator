Footer 01

About

This footer block should be used only once. It has a main description on the left side and social icons on the right side with cool hover effects. You will see this footer block in action inside "Layout 1".

You can control the animation of each block by adding a class name.

These are all the options available for the animations: ".slide-left", "slide-right", "slide-top", "slide-bottom".

However, there are more options avaliable, you can control the speed, the distance and more...

Usage

To use this block you have to copy and paste the following HTML code from here or from the "ui-kit" folder: 


 <footer class="footer-01">
    
 
      <div class="container-fluid">
       
       <div class="row">
           <div class="contact-details col-md-6">
             <div class="content">
             <h1>GREAT SUPPORT</h1>
              <p>We love what we did and we want this to be used by as many people as possible. Therefore, we offer you our help solving whatever problem you might encounter.
              </p>
              <a class="btn btn-hallow btn-lg" href="#">Contact Me</a> 
           </div>
           </div>
           
           <div class="contact-social col-md-6">
             
             <div class="row text-center">
               <div class="social-icon col-md-6">
                
                <a href="#"><i class="fa fa-facebook fa-4x"></i></a>
               
               </div>
               <div class="social-icon col-md-6">
                <a href="#"><i class="fa fa-google-plus fa-4x"></i></a>

               </div>
               <div class="social-icon col-md-6">
                <a href="#"><i class="fa fa-twitter fa-4x"></i></a>

               </div>
               <div class="social-icon col-md-6">
                <a href="#"><i class="fa fa-behance fa-4x"></i></a>

               </div>
             </div>

           </div>
           

       </div>
       


    
      </div>

    

  </footer>

How to modify

All the footers blocks are based on Bootstrap grid system. You can modify any element of this content block, but we suggest that you should keep the same HTML structure if you want to keep the same look and functionality.

If you want to modify the description you should look for <div class="contact-details col-md-6"> and for modifying the social icons <div class="contact-social col-md-6">

If you want to change the background color for the description you should look for .footer-01 {...}.

If you want to change the background color for the social icons you should look for .footer-01 .contact-social .social-icon:nth-child(1). :nth-child(1) means that you can change the background color for the first social icon. If you want to change the second one you should write :nth-child(2) and so on...

Footer 03

About

This footer block can be used multiple times. It has a logo, title and a form input for subscribing to a newsletter. You will see this footer block in action inside "Layout 4" and "Layout 5".

You can control the animation of each block by adding a class name.

These are all the options available for the animations: ".slide-left", "slide-right", "slide-top", "slide-bottom".

However, there are more options avaliable, you can control the speed, the distance and more...

Usage

To use this block you have to copy and paste the following HTML code from here or from the "ui-kit" folder: 


<footer class="footer-03">

    <div class="container">
      <div class="row">
        
        <div class="title text-center">
          <a class="logo" href="#"><img src="../../common/images/logo-footer-03.png" alt=""></a>
          <h2>Subscribe our newsletter</h2>

        </div>
        </div>
         
      </div>
         
        <div class="subscribe">
         
         <div class="container">
           
        <div class="row">
         
         <form name="subscribe" id="newsletter" novalidate="">
            <div class="col-md-9">
            <div class="form-group">
                <span class="icon"><i class="fa fa-envelope-o fa-lg"></i></span>
                <input type="email" class="form-control" placeholder="Your Email *" id="email" required="" data-validation-required-message="Please enter your email address." aria-invalid="false">

How to modify

All the footers blocks are based on Bootstrap grid system. You can modify any element of this content block, but we suggest that you should keep the same HTML structure if you want to keep the same look and functionality.

If you want to change the background color for the description you should look for ..footer-03 .subscribe {...}.

If you want to change the background color for the social icons you should look for .footer-01 .contact-social .social-icon:nth-child(1). :nth-child(1) means that you can change the background color for the first social icon. If you want to change the second one you should write :nth-child(2) and so on...
