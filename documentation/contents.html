Content 01

About

This content block can be used multiple times.It has a smooth fade in animation and some cool effect for the buttons. You will see this content block in action inside "Layout 1".

You can control the animation of each block by adding a class name.

These are all the options available for the animations: ".slide-left", "slide-right", "slide-top", "slide-bottom".

However, there are more options avaliable, you can control the speed, the distance and more...

How to modify

All the content blocks are based on Bootstrap grid system. You can modify any element of this content block, but we suggest that you should keep the same HTML structure if you want to keep the same look and functionality.

It contains two main rows and each one contains 2 columns with a description and an image. If you want to modify the description you should look for <div class="description col-md-5" > and for modifying the image <div class="image col-md-7">

For example if you want the block to appear from the left side you should add this ".slide-left". So let's say we want to add this effect to the description block: <div class="slide-left description col-md-5" >

You can control also the speed and the distance for the animation:

<div class="slide-left description col-md-5" data-plugin-options='{"speed":200, "distance":400}' >

How to modify

All the content blocks are based on Bootstrap grid system. You can modify any element of this content block, but we suggest that you should keep the same HTML structure if you want to keep the same look and functionality.

It contains one row and each one contains 3 columns with an big SVG icon, title and description. If you want to modify the content you should look for <div class="item col-md-4">

If you want to change the SVG icon you should look for: 

             
        <!-- SVG ICON -->         
            <div class="icon">
              <div class="iconmelon">
                  <svg viewBox="0 0 32 32">
                    <g filter="">

                    </g>
                  </svg>
                </div>
            </div>

        <!-- END SVG ICON -->
