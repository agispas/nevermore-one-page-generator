About

You can control the animation of each block by adding a class name.

However, there are more options avaliable, you can control the speed, the distance and more...

How to modify

If you want to change the slider items you should look for <div id="portfolio-1" class="owl-carousel owl-theme"> and <div id="portfolio-2" class="owl-carousel owl-theme"> and from there it's pretty straightforward.

Inside <div class="owl-carousel owl-theme"> </div> are the carousel items like images,videos...

You can control some settings/effects via "main.js" file which you can find inside "common > js" folders. Here are the settings for the carousel:

("#portfolio-1").owlCarousel({
    items:1,
    autoHeight:true,
    dots:false,
    nav:true,
    navText:['‹','›']

   
});
We strongly suggest to not modify any of these settings unless you have some coding experience because it may break the layout.

However, if you want to learn more about the carousel functionality and other options you can check out their website for more details.

  -webkit-transition: opacity 0.3s;

If you want to change the image and the content you should look for <div class="col-md-6 picture-item"> and there you will find the img element for changing the image and the figcaption element for changing the hover details.
