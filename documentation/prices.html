Prices 01

About

This price block can be used multiple times. It contains 3 columns per row. Each columns has an title, descrpition and a cool border hover effect. You will see this price block in action inside "Layout 1" and "Layout 4".

How to modify

If you want to modify the content of the features you should look for <ul class="pricing-features">.

If you want to change the color of the border you should look for .box svg line { ... stroke: #aba197;}
