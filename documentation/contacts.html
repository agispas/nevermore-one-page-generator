Contacts 01

About

This contact block can be used multiple times. It contains a contact form, full width google map and a description. You will see this contact block in action inside "Layout 1" and "Layout 6".

You can control the animation of each block by adding a class name.

These are all the options available for the animations: ".slide-left", "slide-right", "slide-top", "slide-bottom".

However, there are more options avaliable, you can control the speed, the distance and more...

How to modify

If you want to modify the content of the contact details you should look for <div class="contact-details">.

If you want to change the content of the contact form you should look for <form name="sendMessage" id="contactForm" novalidate="">

If you want to change the content of the social icons you should look for <ul class="social-icons text-center">. The icons used are from "font-awesome". You can see more details on their website.

To change the map you have to go to the google maps website and from there you choose the location and click the bottom right "settings" icon and choose embed map and from there copy the iframe code here <div class="map">
