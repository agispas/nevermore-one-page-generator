Teams 01

About

This teams block can be used multiple times. It contains one slider and description blocks for each team member. You will see this team block in action inside "Layout 2".

You can control the animation of each block by adding a class name.

These are all the options available for the animations: ".slide-left", "slide-right", "slide-top", "slide-bottom".

However, there are more options avaliable, you can control the speed, the distance and more...

How to modify

If you want to change the slider items you should look for <div class="item">. There you will find two columns each one containing an image, a description and a button.

You can control some settings/effects via "main.js" file which you can find inside "common > js" folders. Here are the settings for the carousel:

$('#team-01').owlCarousel({
 
    items:1,
     dots:true,
    dotsContainer:'.dots',
   

  });
We strongly suggest to not modify any of these settings unless you have some coding experience because it may break the layout.

However, if you want to learn more about the carousel functionality and other options you can check out their website for more details
