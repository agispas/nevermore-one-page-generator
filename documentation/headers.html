How to modify

Inside the <div class="jumbotron">...</div> you can modify the existing content with whatever you want.

Inside <div id="header-1-carousel" class="owl-carousel owl-theme"> </div> are the carousel items like images,videos...

You can control some settings/effects via "main.js" file which you can find inside "common > js" folders. Here are the settings for the carousel:

$("#header-1-carousel").owlCarousel({
    items:1,
    video:true,
    nav:false,
    videoHeight:600

});
We strongly suggest to not modify any of these settings unless you have some coding experience because it may break the layout.

However, if you want to learn more about the carousel functionality and other options you can check out their website for more details.

If you want to change the background color of the <div class="jumbotron">...</div> You will find the style inside header-01.css or header-01.less: 

  .header-01-sub .jumbotron {
  margin: 0;
  padding-top: 98px;
  padding-bottom: 98px;
  background: #bebaae;
}

If you want to change the background color look for .jumbotron class and from there it's pretty straightforward.

How to modify

Inside the <div id="header-2-carousel">...</div> you can modify the existing content with whatever you want.

Inside the <div class="item ratio">...</div> you can modify the image and the caption for each slide.

You can control some settings/effects via "main.js" file which you can find inside "common > js" folders. Here are the settings for the carousel:

$("#header-2-carousel").owlCarousel({
    items:1,
    animateOut: 'fadeOut',
   

  
});

"fadeOut" value is the only built-in CSS animate style. However there are tons of additional CSS animations that you can use in Owl. Simply download animate.css library and you are ready to extend Owl with new fancy transitions.

We strongly suggest to not modify any of these settings unless you have some coding experience because it may break the layout.

However, if you want to learn more about the carousel functionality and other options you can check out their website for more details

You can easily modify how the navigation will behave via "main.js" file:

    menuWidth: '22.625em' 
      });
We strongly suggest to not modify any of these settings unless you have some coding experience because it may break the layout.
